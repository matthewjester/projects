# Poet Battle Bot

Telegram bot for poetry battles — voting, rating, leaderboards.

Built for [@baliaicommunity](https://t.me/baliaicommunity)

## Stack
- Node.js + node-telegram-bot-api
- Notion API (database backend)
- Deployed via PM2

## Setup
```bash
export BATTLE_BOT_TOKEN=your_telegram_bot_token
export BATTLE_NOTION_KEY=your_notion_api_key
npm install
node bot.js
```
