# poet-battle-bot

Telegram-бот (@poetbattlebot) для голосования за артистов на Poet Not Dead Battle.

## Функционал

### Для пользователей:
- Выбор категории (Поэзия / Проза / Музыка)
- Просмотр артистов с ссылками на произведения
- Голосование (по 1 голосу в каждой категории)
- Возможность переголосовать

### Для админов (@poetnotdead creator + admins):
- `/stats` — статистика голосования
- `/export` — экспорт результатов (JSON + текст)
- `/close` — закрытие голосования

## Notion интеграция

**3 базы данных:**

1. **Артисты** (306a74b651a8817ea7d6ecb45d706eed)
   - № (Number)
   - Имя артиста (Title)
   - Название произведения (Text)
   - Ссылка (URL)
   - Категория (Select: Поэзия / Проза / Музыка)
   - Активен (Checkbox)
   - Голосов (Number)

2. **Голоса** (306a74b651a8816493dcefc530542f8a)
   - ID (Title)
   - User ID (Number)
   - Username (Text)
   - Имя пользователя (Text)
   - Категория (Select)
   - Артист (Relation → Артисты)
   - Дата/время (Date)

3. **Настройки** (306a74b651a881839f5bc8776481a934)
   - Параметр (Title)
   - Номер баттла (Number)
   - Статус (Select: Открыто / Закрыто)
   - Welcome текст (Text)

```bash

```

## Логика работы

1. Пользователь открывает `/start`
2. Проверка статуса голосования (Открыто/Закрыто)
3. Если закрыто → сообщение с ссылками на соцсети
4. Если открыто → выбор категории
5. Список артистов (кнопки с ссылками + кнопки голосования)
6. При голосе:
   - Проверка существующего голоса
   - Обновление или создание записи в "Голоса"
   - Инкремент/декремент счётчика у артистов
7. Подтверждение голоса
8. Кнопка "Главное меню"

## Админские команды

### `/stats`
Показывает статистику по всем категориям:
- Список артистов с количеством голосов
- Сортировка по убыванию голосов
- Общее количество проголосовавших

### `/export`
Экспортирует результаты:
- JSON файл с детальными данными
- Текстовое сообщение со статистикой

### `/close`
Закрывает голосование:
- Обновляет статус в базе "Настройки"
- После этого пользователи видят сообщение с соцсетями

## Переменные окружения

- `BOT_TOKEN` — токен @poetbattlebot
- `NOTION_KEY` — Notion API key
- `DB_ARTISTS`, `DB_VOTES`, `DB_SETTINGS` — ID баз данных
- `ADMIN_GROUP_ID` — ID группы @poetnotdead для проверки админов
